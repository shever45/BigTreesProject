# BigTreesProject

Прогнозирование оттока клиентов с помощью алгоритмов дерева решений

Построить и сравнить три модели машинного обучения на основе алгоритмов дерева решений для прогнозирования оттока клиентов компании. Основная задача — cоздать модель предсказания того, уйдёт ли абонент в отток (churn).
