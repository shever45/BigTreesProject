# Прогнозирование оттока клиентов телеком-компании
# Цель
Разработать моделей машинного обучения для предсказания оттока клиентов на основе данных о клиентах телеком-компании.

# Описание проекта
В ходе проекта выполнен полный цикл анализа данных и построения ML-моделей:

EDA и feature engineering — исследование 7,032 записей с 21 признаком, создание сегментов (когорт) по длительности обслуживания

Визуализация — анализ распределения признаков и их корреляции с оттоком

Моделирование — реализованы и сравнены 4 алгоритма на основе деревьев:

**Decision Tree**

**Random Forest**

**AdaBoost**

**Gradient Boosting**

# Результаты — лучшая модель (AdaBoost) достигла F1-score 0.57 для класса "отток" при общей accuracy 83%

# Технологии
Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebook

# Ключевые инсайты
Наиболее значимые признаки: тип контракта, наличие техподдержки, интернет-сервисы

Клиенты с помесячным контрактом уходят в 3 раза чаще

Создание когорт по длительности обслуживания улучшило интерпретируемость модели

# Итог 
**Создан рабочий прототип системы прогнозирования оттока с рекомендациями для отдела удержания клиентов.**
